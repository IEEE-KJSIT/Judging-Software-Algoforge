# Judging-Software-Algoforge

Hackathon judging app for **AlgoForge** — admin controls active team, judges score on mobile, live leaderboard + CSV export.

**Stack:** React, TypeScript, Vite, Tailwind, Firebase (Auth + Firestore), Vercel-ready.

See **[SETUP.md](./SETUP.md)** for Firebase setup, local dev, and Vercel deployment.
